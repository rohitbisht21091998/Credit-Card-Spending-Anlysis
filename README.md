# Credit Card Spending Analysis

## About
Analysed 26,000+ credit card transactions across 
cities, categories, card types and gender using 
Excel pivot tables and dashboard.

## Key Insights
- Bills is top spending category at 22%
- Female customers contribute 54% of total spend
- June-July lowest spending months
- October spike due to festive season
- Top 4 cities contribute majority of spend
- 84% transactions are High value (above ₹20,000)
- All card types show similar spending levels

## Tools Used
Microsoft Excel — Pivot Tables, Charts, Dashboard

## Dataset
Source: Kaggle — Credit Card Transactions India
Rows: 26,052
